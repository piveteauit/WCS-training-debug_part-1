# Checkpoint 2

## Step 1 - Start the project

1. Clone the repo from Github.
2. Create a branch with your name and switch on it.
3. Run `npm run setup`.
4. Run `npm run dev`.
5. Go to `localhost:3000` with your favorite browser.
